# 🇪🇸 Spain Immigration Guide

Plataforma web estática desenvolvida com **HTML5 + TailwindCSS**, focada em fornecer informações organizadas para brasileiros que desejam morar na Espanha.

O projeto é responsivo, leve e otimizado para SEO.

---

## 📌 Objetivo

Centralizar informações sobre:

* Cidades da Espanha
* Tipos de visto
* Documentação necessária
* Custo de vida
* Mercado de trabalho
* Comparação Espanha x Brasil

---

## 🧰 Stack Tecnológica

* HTML5
* TailwindCSS
* JavaScript (opcional para interações)
* CLI
* Node.js (para build do Tailwind)

## ⚙️ Instalação

### 1️⃣ Clonar o projeto

```bash
git clone https://github.com/seu-usuario/spain-immigration-guide.git
cd spain-immigration-guide
```

### 2️⃣ Instalar dependências

```bash
npm install
```

### 3️⃣ Rodar Tailwind em modo watch

```bash
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
```
